# Laisky MCP API Catalog

The RFC 9727 API catalog is available at [https://mcp.laisky.com/.well-known/api-catalog](https://mcp.laisky.com/.well-known/api-catalog).

It links agents to the Laisky MCP OpenAPI description, MCP discovery endpoint, MCP server card, authentication guide, status endpoint, and GraphQL playground.

Important entries:

- [OpenAPI service description](https://mcp.laisky.com/openapi.json)
- [MCP discovery endpoint](https://mcp.laisky.com/.well-known/mcp)
- [MCP server card](https://mcp.laisky.com/.well-known/mcp/server-card.json)
- [Authentication guide](https://mcp.laisky.com/auth.md)
- [Status endpoint](https://mcp.laisky.com/status)
